## 新增功能模块规划

### 1. 资源管理模块
- 资源上传（支持图片/PPT/Word/PDF/视频等多种格式）
- 资源列表（网格/列表视图切换）
- 资源预览（图片预览、PDF阅读、视频播放）
- 资源分类和标签管理
- 资源搜索和筛选

### 2. 知识库管理模块
- 知识库创建和配置
- 资源索引管理（关联文章、资源到知识库）
- 知识库内容搜索
- 知识图谱可视化（简化版）
- 索引状态监控

### 3. AI 对话模块
- 基础 LLM 对话
- 基于项目知识库的问答
- 结合网络搜索的回答
- 对话历史管理
- 引用来源展示

### 4. 网络爬虫/搜索模块
- URL 抓取（输入网址获取内容）
- 网络搜索（关键词搜索）
- 内容摘要（LLM 自动总结）
- 抓取结果保存到知识库

## 技术方案

- **路由**: 基于 `location.hash` 的简易 SPA 路由
- **模块化**: 使用 ES Module (`type="module"`)，live-server 天然支持
- **样式**: CSS 变量定义主题，模块化拆分
- **数据**: 全部使用 mock 数据，模拟异步加载
- **交互**: 原生 DOM 操作 + 事件委托，不依赖任何框架
- **图标**: 内联 SVG sprite
